# PyraTools - Premium Client-Side Media Tools Hub

PyraTools is a 100% private, client-side, free utility hub offering 10+ browser-based tools for image conversions, PDF modifications, OCR text extraction, and social media text styling. 

Designed with a premium light theme, optimized mobile responsiveness, and zero server uploads.

## 🛠️ Features

*   **Image Converters & Compressor**: HEIC to JPG/PNG, WebP Converter, SVG to PNG rasterizer, and Smart Image Compressor.
*   **PDF Utilities**: Image to PDF compiler with custom sorting, PDF to Image page extractor, and PDF size compressor.
*   **OCR Text Scanner**: Scrapes editable text content from images using Tesseract OCR.
*   **Social Formatting**: Unicode Fancy Fonts Generator and Instagram Bio Spacer line breaker.

## ⚡ Technical Architecture

*   **Language**: Vanilla HTML5, CSS3, ES6 JavaScript.
*   **Client-Side Execution**: All computations occur locally in the user's browser (zero server hosting fees, 100% data privacy).
*   **Libraries**: PDF.js, jsPDF, heic2any, JSZip, and Tesseract.js.
*   **SEO Optimization**: Dedicated pages per tool, semantic tags, automated sitemap.xml, and structured JSON-LD schemas.
