SVG to PNG Converter FAQ

What is an SVG file?
SVG (Scalable Vector Graphics) is an XML-based vector image format. Because it uses mathematical formulas rather than pixels, it can scale to any size without losing quality. PNG and JPG are raster formats that use pixels.
How does SVG to PNG rendering work in the browser?
The browser natively knows how to render SVG files. The JavaScript reads the SVG file, converts it to an XML data URL, loads it inside an image element, draws it onto a canvas scaled up by your multiplier choice, and exports it as a raster PNG or JPG.
Is the vector background transparency preserved?
Yes. If you choose the PNG format, the output image will maintain transparent sections of the original SVG. Choosing JPEG will automatically fill transparent areas with a solid white background.
