Image Compressor FAQ

How does in-browser image compression work?
Our tool uses the HTML5 Canvas API inside your web browser. When you load an image, JavaScript draws it onto a canvas and extracts a compressed version using custom quality parameters locally. Your image is never uploaded to a server.
Will compressing my image reduce its quality?
Yes, but you can control the compression level. Reducing quality to 70-80% will significantly shrink file size while remaining virtually identical to the human eye (lossless compression effect).
Is there a file size limit?
The tool can comfortably compress images up to 15MB. Because processing is done using your computer's RAM, very large images might take a few seconds to process depending on your browser.
Are my images saved on your servers?
Absolutely not. PyraTools processes everything locally. Once you download the compressed images or close the tab, the images are cleared from your browser memory.
