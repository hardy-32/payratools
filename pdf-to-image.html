PDF to Image FAQ

How does browser-side PDF extraction work?
We use Mozilla's PDF.js library via CDN. It parses the PDF binary data inside your browser, draws each page vector text and layouts onto an HTML5 Canvas at your chosen scaling factor, and exports the canvas as a standard PNG or JPEG image.
Why download as a ZIP file?
If your PDF has multiple pages, downloading them individually would prompt you for multiple saves. The ZIP option uses `JSZip` library client-side to package all extracted page images into a single ZIP file download, saving time and keeping files organized.
Will my document's formatting remain intact?
Yes. PDF.js is highly accurate at rendering PDF layouts, fonts, images, and vectors exactly as they appear in Acrobat or other standard PDF readers.
